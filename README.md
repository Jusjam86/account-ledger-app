# Account Ledger Application
### Description
This Java terminal application allows you to track all financial transactions for abusiness or for personal use and is saved locally onto a _transactions.csv file_ that you can access and see through the terminal.
## Features
- **Deposit**: prompt user for the credit information and saves it to the csv file
- **Payment**: prompt user for the debit information and saves it to the csv file
- **View Ledger**: allows user to look through various screens to view either all or selected transactions
- **Report**: a search that filters your transactions by date or vendor
